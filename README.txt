逃げ恥人物判定

<概要>
アップロードした画像に選択した人物が写っているか判定します。

写っていた場合は判定した顔を四角で囲い表示します。
判定できない場合は　"検出できませんでした"　と表示されます。

画像が小さすぎたり顔が正面を向いていないと上手く認識できないことがあります。

<ファイル詳細>
face_cut.py: open.cv で"haarcascade_frontalface_alt2.xml"を使って元画像から顔を切り出すプログラム
　　　　　　　切り出した画像は facedct に保存される
　　　　　　　この時点で不要な画像の削除を行ったほうが良い

increase.py: 切り出した顔画像を加工し画像の水増しを行うプログラム
　　　　　　　加工した画像は increse フォルダに保存される

facerecg4_train.py: 加工した画像を用いてvgg16を利用し特徴量を学習するプログラム

pred2.py: Flaskから画像データを受け取り顔を切り出し学習させたデータから人物の判定を行うプログラム

app_nigehaji.py: Flask
